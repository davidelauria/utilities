Uso di Stream TCP in ambiente Windows

......... servTCP.c ......... server TCP, C source
......... servTCP.dsp ..... server TCP, Msvc 5.0 project
......... servTCP.dsw ..... server TCP, Msvc 5.0 workspace
......... cliTCP.c ............ client TCP, C source
......... cliTCP.dsp ......... client TCP, Msvc 5.0 project
......... cliTCP.dsw ........ client TCP, Msvc 5.0 workspace

Scopo:
Il client richiede la connessione al server e gli spedisce una stringa.
Il server riceve la stringa, ne trasla i caratteri di 2, e rispedisce al client.
Il client riceve la stringa traslata e la stampa.

N.B. USO
Collocare i due file in due host anche non diversi, ( es.: servTCP.c in host1 e cliTCP.c su host2 ),
compilare,
eseguire prima servTCP su host1 (es: servTCP 5001),
e poi eseguire cliTCP su host2 (es: cliTCP IP_host1 5001 ),

N.B. Compilazione e Linking
--- su Windows 95 --- usare la libreria wsock32.lib
--- su Windows NT / Windows 2000 --- usare la libreria ws2_32.lib
